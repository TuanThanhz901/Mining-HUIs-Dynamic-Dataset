Introduction

Welcome to the High Utility Itemset Miner! This application is designed to help researchers, data analysts, and business professionals efficiently mine high utility itemsets from data.

This website was developed by Le Tuan Thanh and Pham Hoang Trung Kien, a student from the Faculty of Information Technology, Ton Duc Thang University. It is a demo project for the Information Technology Project course, under the guidance of Mr. Doan Xuan Thanh.

The goal of this website is to explore high occupancy itemsets by using the IMEFIM algorithm:

  • Interleaved pattern mining with Efficient Frequent Itemset Mining (IMEFIM).

IMEFIM was introduced by Loan T.T. Nguyen and other co-authors in 2019. It utilizes advanced interleaving techniques to discover high occupancy itemsets efficiently. However, you may encounter difficulties in selecting the most suitable minimum occupancy threshold to determine the correct number of itemsets.

Provide

The website provides the following functions:

  • Mine high occupancy itemsets using the IMEFIM algorithm.
  • Allow users to upload datasets and adjust utility thresholds.
  • Display and interpret mining results for analysis.

Format of source file

The input file is a text file containing transaction data. Each line represents a transaction, where items are separated by spaces, followed by the total utility value (profit) separated by a colon. The individual utilities of items in the transaction are also separated by spaces and are listed after the total utility, separated by another colon.

Format of returned file

The returned file is a text file containing the high occupancy itemsets mined from the input data. Each line represents an itemset and its corresponding utility value, separated by a space. Items in the itemset are enclosed in square brackets and separated by commas.
